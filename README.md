## Instalation

### Manager
1. Clone repository
```sh
$ git clone https://github.com/jara155/terminal.git
```
2. Update permisions to execute
```sh
$ chmod +x manager.sh
```
3. Run script manager.sh
```sh
$ ./manager.sh
```
